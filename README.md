# 🎀 hello-kitty-guests

Фанатский проект, где игроки могут делиться своими яркими моментами из игры **Hello Kitty Island Adventure**. Это одностраничное веб-приложение с возможностью загрузки изображений и их просмотра в галерее. Создан в учебных целях.
